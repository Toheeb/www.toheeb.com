Base CSS

Base helps CSS Authors create Personalizable Web Design in 3 steps:

  1. Link the classless stylesheet from a well-structured markup.

  2. Extend with custom stylesheets.

  3. Add the

At its core, Base resets the browser's stylesheet beyond blank or normal. It ups to a state ready for reading, customizing, and personalizing.

Installation Code

on all elements

Base does 2 things on all standard and custom elements plus thier pseudos:

  1. Accept a value to soften the corners of all bordered elements

  2. Make box sizing intuitive

Universal Code 
/* 1 */
:root {
  --base-corner-radius: .2em;
}

*, ::before, ::after 
{
  /* 1 */
  border-radius: var(--base-corner-radius);

  /* 2 */
  box-sizing: border-box;
}

on typography

Base does 6 things on typography of elements:

  1. Accept font family, font size, font scale, and line height for customizations and personalizations.

  2. Generate an 11-item geometric sequence of sizes using the base font size and font scale.

  3. Apply the respective size on: a, Kickers and Micros; b, Containers, Paragraphs, and Titles; c, Subheaders; and d, Heading Level 6 to 1.

  4. On containers, blocks, and micros, Base furthers with the following:

    1. Typeface them with the font family.

    2. Root-em their respective font size.

    3. Reduce their line height as font size gets larger.

  5. Hint at removing orphans in plain paragraphs

  6. Hint at balancing words of each line in lead blocks.

on spacing

Base does 11 things for vertical and horizontal spacing:

  1. Accept an indent mode, a block gap, and a line length for customizations and personalizations.

  2. Redefine block gap to increase or decrease with respect to font size or line height.

  3. Generate a 9-item arithmetic sequence of block gaps that responds to indent mode

  4. Fix descender overflows of heading blocks (at larger sizes.)

  5. Use the sequence to create vertical space hierarchy across: a, Kickers; b, Subheaders; c, Paragraphs and Mixes; d, Titles; e, Parts; and f, Sections;

  6. Allow partial or full-line indent on the right paragraphs

  7. Cap width of contents to the given line length value and center them.

  8. Cap width of blockable media to their parent to avoid overflow; and center them.

  9. Fluid and center Table.

  10. Vertically Align inline media to surrounding text.

  11. Support up to 3-digit markers

on personalization

Base helps authors with a Settings Component. This component provides an interface for users to personalize their experience of the author's design as follows:

  • Change the face of all text to any font of their choice.

  • Bump and scale sizes to the preference of their sight

  • Breathe spaces inside, between, and around blocks.

  • Theme page in light and dark mode to their favourite color.

  • Switch from custom layout to read page in document layout with classic or modern indent.
